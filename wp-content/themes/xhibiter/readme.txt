== Description ==

Xhibiter is a cutting-edge WordPress theme designed specifically for creating NFT marketplaces, AI tools, DAO, ICO landings, Crypto, and Business websites. With its seamless integration with WooCommerce and support for Elementor, you can easily build and customize your own website. The theme also features a sleek dark mode design, giving your website a modern and sophisticated look.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
